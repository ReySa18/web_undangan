# 🎵 Folder Musik Background

Letakkan file musik Anda di folder ini dengan nama:

```
background.mp3
```

Contoh path lengkap:
```
public/music/background.mp3
```

## Rekomendasi
- Format: **MP3** (paling kompatibel di semua browser)
- Durasi: minimal 3-5 menit, atau gunakan lagu yang akan di-loop otomatis
- Bitrate: 128kbps sudah cukup (filenya tidak terlalu besar)
- Volume default sudah diset ke 50% secara otomatis
